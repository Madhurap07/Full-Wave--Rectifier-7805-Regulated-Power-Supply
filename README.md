# Full-Wave--Rectifier-7805-Regulated-Power-Supply
Designed and fabricated a Full-Wave Rectifier with 7805 regulated power supply. The project includes Proteus schematic design, PCB layout development, manual PCB fabrication, drilling, and soldering.

The complete workflow includes:
- Circuit schematic design in Proteus
- PCB layout generation
- PCB fabrication on copper-clad board
- Drilling and manual soldering
- Hardware testing and verification

## Features
- Full-Wave Center-Tapped Rectifier
- 7805 Voltage Regulation
- LED Power Indicator
- Single-Layer PCB Design
- Through-Hole Components

## Tools Used
- Proteus Design Suite
- PCB Fabrication Process
- Soldering Techniques
